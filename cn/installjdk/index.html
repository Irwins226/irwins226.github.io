在Centos操作系统中安装和配置jdk

Cenos 操作系统安装jdk1.8

1 检查并卸载 OpenJDK

查看 Java 相关信息，检查系统是否自带了 OpenJDK 以及相关安装包，如果有的话则应先将其卸载

java -version

rpm -qa | grep java

// rpm（RedHat Package Manger）：RedHat软件管理工具，是一种用于打包及安装的工具
// -qa（query，all）
// grep（global search rgular expression(RE) and print out the line）：是一种强大的文本搜索工具

由于本人的服务器操作系统并没有安装Java环境，我都是直接使用docker进行管理的，所以执行如下

[root@VM-0-13-centos jar]# java -version

-bash: java: command not found

[root@VM-0-13-centos jar]# rpm -qa | grep java

执行【rpm -qa | grep java】如果执行为空白，说明本地并没有安装Java相关

如果已经安装了想卸载重装可以使用命令 rpm -e --nodeps ，如：

rpm -e --nodeps tzdata-java-2012c-1.el6.noarch

rpm -e --nodeps java-1.6.0-openjdk-1.6.0.0-1.45.1.11.1.el6.x86_64

// -e：删除指定的套件
// --nodeps：不验证套件档的相互关联性
